How to use

Please refer to the README file on GitHub to get the latest info on how to use this plugin.

The easiest way to install Hongkong is with Bower.

npm install --save hongkong

The scripts needed: jQuery, a requestAnimationFrame polyfill and Hong Kong itself. Letâ€™s say you installed Hong Kong with Bower:

<script src="node_modules/jquery/dist/jquery.js"></script>
<script src="node_modules/hongkong/hongkong.js"></script>

Just add an attribute data-parallax-top or data-parallax-bottom to the element you want to be animated in the named direction.

You can add a data-parallax-factor with a value which is an integer to change the factor of which the element should move. Default is 4.
